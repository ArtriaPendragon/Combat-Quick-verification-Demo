# 怪物仇恨与多人目标选择规则

最后更新：2026-05-10

## 1. 当前目标

怪物目标选择以仇恨表为核心。每个候选玩家或可攻击单位对应一个仇恨条目，怪物优先攻击仇恨最高且有效的目标。

## 2. 仇恨条目

| 字段 | 用途 |
| --- | --- |
| 目标 | 玩家角色或可攻击单位。 |
| 仇恨值 | 决定目标优先级。 |
| 最近更新时间 | 后续用于衰减或调试。 |

## 3. 仇恨变化来源

- 玩家造成伤害。
- 玩家靠近怪物。
- 玩家破坏部位。
- 玩家治疗或支援队友。
- 行为树脚本事件。

## 4. 目标切换规则

- 如果当前没有目标，选择最高仇恨目标。
- 如果新目标仇恨超过当前目标加上切换阈值，允许切换。
- 切换阈值来自 `UMonsterAIDataAsset::TargetSwitchHatredThreshold`。
- 目标无效时清理目标并回到搜索或巡逻。

## 5. 多人影响

`FMonsterAttackScoreContext::ActivePlayerCount` 会进入招式评分。范围攻击、扫尾、吼叫等招式可以通过 `MultiTargetBonus` 提高多人战斗中的分数。

## 6. 必要日志

- 仇恨增加。
- 当前目标切换。
- 最高仇恨目标选择。
- 目标无效原因。
- 当前玩家数量。
