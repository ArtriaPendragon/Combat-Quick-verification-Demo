# 增强输入资产占位文档

最后更新：2026-05-10

## 1. 当前定位

本文档属于历史玩家输入系统。当前怪物 AI 标准不依赖这些输入资产，但工程仍保留玩家输入控制器和增强输入软引用。

## 2. 建议目录

```text
/Game/CombatAgent/Input
```

## 3. 建议资产

| 用途 | 资产类型 | 建议名称 |
| --- | --- | --- |
| 默认输入映射上下文 | 输入映射上下文 | `IMC_CombatAgent_Default` |
| 移动 | 输入动作 | `IA_Move` |
| 镜头 | 输入动作 | `IA_CameraLook` |
| 轻攻击 | 输入动作 | `IA_LightAttack` |
| 闪避 | 输入动作 | `IA_Dodge` |
| 锁定 | 输入动作 | `IA_LockOn` |
| 左切换目标 | 输入动作 | `IA_SwitchTargetLeft` |
| 右切换目标 | 输入动作 | `IA_SwitchTargetRight` |

## 4. 规则

- 源码中不硬编码这些资产路径。
- 玩家控制器只暴露可配置软引用。
- 实际资产引用应通过类默认值、蓝图子类或配置文件注入。
- 该文档不影响怪物 AI 数据资产、行为树、黑板或 EQS 的创建。
